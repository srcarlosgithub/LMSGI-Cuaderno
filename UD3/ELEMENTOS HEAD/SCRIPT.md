#  **CABECERA HTML**

## - **Script** 

Esta etiqueta **permite añadir un Script** de código ejecutable, como puede ser **JavaScript**. Puede contener el código o referenciar a un *recuerso externo*

**EJEMPLO**:
```HTML
<html>
 <head>
 <script> alert(“hola”); </script>
 </head>
…
<script src=”js/main.js” />
```
